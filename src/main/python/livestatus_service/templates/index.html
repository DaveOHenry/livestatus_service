Livestatus service

The livestatus-service exposes the functionality of a livestatus socket to the outside world over HTTP.

  • Performing queries

    GET /query?q=QUERY

    Will perform a query using QUERY. An easy example would be /query?q=GET%20hosts

    If you need newlines, e.G. to add a filter, use \n.

  • Performing commands

    GET /cmd?q=COMMAND

    Will perform a command using COMMAND. The COMMAND [%s] directive will be inserted for you. An easy example would be /cmd?q=DISABLE_HOST_SVC_CHECKS;devytc
